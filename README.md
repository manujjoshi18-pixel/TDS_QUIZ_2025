# IITM LLM Analysis Quiz Project

This repository contains the solution for IITM's LLM Analysis Quiz.

## Features
- Flask REST API at `/task`
- Solves quiz tasks using Playwright (JavaScript rendering)
- PDF table extraction using pdfplumber
- HTML table extraction
- Supports ngrok for public endpoint
- Fully compatible with GitHub Codespaces
